## Run the info site of starswap

```
npm install
npm run dev
```

Open Chrome and go to `http://localhost:9528/`

## Build the info site of starswap

```
npm run build:prod
```

## Deploy the info site of starswap

```
git tag vx.x.x
git push --tag
```

`vx.x.x` means the latest version, such as `v0.1.1`
